## Parse(string) {#parse_1}

Converts the string representation of an UID to its typed equivalent.

```csharp
public static Uid Parse(string s)
```

| Parameter | Type | Description |
| --- | --- | --- |
| s | String | A string containing an UID to convert. |

### Return Value

A typed [`Uid`](../../uid) equivalent to the UID contained in *s*.

### Remarks

If the requested UID is not found in the list of registered UIDs, a new unknown [`Uid`](../../uid) ([`Type`](../type) is Unknown) with [`Code`](../code), [`Name`](../name) based on the given *s* is returned.
