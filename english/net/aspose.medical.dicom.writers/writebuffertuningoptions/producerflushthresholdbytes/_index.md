## WriteBufferTuningOptions.ProducerFlushThresholdBytes property

Number of produced bytes after which a flush is attempted to apply backpressure for synchronous writers. Values are clamped to [`MinFlushThresholdBytes`](../minflushthresholdbytes)…[`MaxFlushThresholdBytes`](../maxflushthresholdbytes) and aligned up to the nearest 4 KiB boundary. Read-only, Int32.

```csharp
public int ProducerFlushThresholdBytes { get; set; }
```

### Remarks

Default value is `64 KiB`.
