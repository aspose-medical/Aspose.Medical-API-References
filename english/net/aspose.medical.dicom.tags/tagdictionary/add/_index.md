## Add(Tag, TagMetadata) {#add}

Adds (registers) the given *tag* and its *metadata* to this tag dictionary. If the given *tag* is not private, then it must be registered in standard dictionary (i.e. [`Default`](../default)). If the given *tag* is private, then it must be registered in private dictionary (i.e. [`TagDictionary`](../../tagdictionary) with not `null`[`PrivateCreator`](../privatecreator)). While registering a tag in private dictionary, the tag's [`Creator`](../../tag/creator) will be updated according to the [`PrivateCreator`](../privatecreator) of this dictionary.

```csharp
public void Add(Tag tag, TagMetadata metadata)
```

| Parameter | Type | Description |
| --- | --- | --- |
| tag | Tag | A tag to be registered. |
| metadata | TagMetadata | Metadata of the given *tag*. |

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Unable to add not private tag into private dictionary. |
