## PhotometricInterpretation.PaletteColor property

Pixel data describe a color image with a single sample per pixel (single image plane). The pixel value is used as an index into each of the Red, Blue, and Green Palette Color Lookup Tables (0028,1101-1103 &amp; 1201-1203). This value may be used only when Samples per Pixel (0028,0002) has a value of 1. When the Photometric Interpretation is Palette Color; Red, Blue, and Green Palette Color Lookup Tables shall be present. See https://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_c.7.6.3.html#sect_C.7.6.3.1.2:~:text=in%20PS3.5%20.-,PALETTE%20COLOR,-Pixel%20data%20describe.
